# Plano Editorial

Estratégia viva. Skill `/post-ideas` lê este arquivo + `calendario.md` antes de propor pauta. Não inventar pilares — usar os 5 já definidos no `calendario.md`.

## Cadência base (semanal)

- **4 posts IG/semana** — segunda, terça, quinta, sexta (qua e fim de semana respiram)
- **1 reel/semana** — preferência sexta
- **1 carrossel longo/mês** — deep-dive em 1 dos 6Ps

## Mix por pilar (mensal)

| Pilar              | %    | Posts/mês (~16 IG) |
| ------------------ | ---- | ------------------ |
| Autoridade técnica | 35%  | ~6                 |
| Anti-guru          | 25%  | ~4                 |
| Provas             | 20%  | ~3                 |
| Atração VSS        | 12%  | ~2                 |
| Atração Advisory   | 8%   | ~1                 |

Mix é referência, não camisa-de-força. Em semana de lançamento, viés VSS sobe pra 30%+.
